주소 기반 테스트 자동화

시작 URL을 넣으면 같은 사이트 안의 페이지를 따라가며 확인하고, 폼이 있는 페이지에서는 빈 제출·값 채움·필수 한 곳만 비우기 같은 제출 시나리오를 자동으로 돌립니다. 그 결과를 숫자로 요약하고, 대시보드·Markdown·JSON 리포트로 남깁니다. 환경에서 Lighthouse를 켜면 크롤 URL 일부에 대해 성능·접근성·SEO 등 점수와 HTML 보고서를 lighthouse/pages/에 둡니다. 로그인·폼 값은 아래 구조화 입력과 request-template.env에 적어 두면 다음 실행에 반영할 수 있습니다.

새 실행

시작 URL로 run 폴더를 만듭니다. 이후 보고서·입력 요청이 그 run에 묶입니다.

값 요청 — 구조화 입력

저장 방식: 아래 입력란에서 「구조화 값 저장」을 누르면 inputs-request.json과 request-template.env가 함께 갱신됩니다. 텍스트 영역만 고친 뒤 「.env만 저장」을 누르면 파일만 바뀌고 JSON 폼과는 어긋날 수 있습니다(고급용).
